# Usermanagement SPA
## Vue3 - Vite - Apollo-Graphql - Tailwind

### Getting Repo
```bash
git clone https://github.com/bmehler/user-management-spa.git
cd user-management-spa
```
### Install Node Modules
```bash
npm install
```
### Starting App
```bash
npm run dev
```
### Open App
```bash
http://localhost:5173/
```
